All real. All built. All free.

7 Phases

DBPF Deep Scanner

Opens compressed .package files other tools can't read. Decompresses binary archives, reads TGI resource tables, and checks them against the live game index.

Binary Format

Crash Predictor

Binary Search

Resource.cfg Validator
